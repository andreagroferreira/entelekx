# EntelekX — Index

EntelekX is a **personal AI operating system**: a self-hosted, adaptive workspace that blends a life assistant, a development studio, a multi-model deliberation engine, and a media forge into one coherent environment.

## Purpose

Help a single person act like a full company by giving them:
- A memory that learns and adapts.
- Tools to build real software.
- Access to the best models without lock-in.
- A private, local-first home for their data and workflows.
