# EntelekX — Phase 3: Dev Studio

## Goal

Turn EntelekX into a real development environment.

## Deliverables

- [ ] Project scaffolding wizards for:
  - Laravel
  - Vue 3
  - Nuxt 4
  - Next.js / React
  - Node.js
  - Python / FastAPI
- [ ] Workspace model and filesystem confinement.
- [ ] Code editor (Monaco or CodeMirror).
- [ ] File tree and diff view.
- [ ] AI edit application with approval.
- [ ] Test/lint runner integration.
- [ ] Git integration:
  - Init, commit, push.
  - Branch creation.
  - PR description generation.
- [ ] Deployment helpers:
  - VPS via SSH.
  - Vercel / Netlify via tokens.
  - Fly / Railway / Render via tokens.
- [ ] Design handoff from Media Forge.

## Out of Scope

- Full IDE features (advanced debugging, LSP).
- All possible deployment targets.

## Definition of Done

A user can scaffold a Nuxt project, edit a file, run tests, commit to Git and deploy to Vercel from inside EntelekX.
