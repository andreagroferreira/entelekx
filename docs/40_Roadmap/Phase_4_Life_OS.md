# EntelekX — Phase 4: Life OS

## Goal

Implement the productivity, business and knowledge management layer.

## Deliverables

- [ ] Notes module:
  - Markdown editor.
  - Backlinks and graph.
  - Tags and templates.
  - AI summarisation and extraction.
- [ ] Tasks module:
  - Create from chat.
  - Projects, priorities, due dates.
  - Recurring tasks.
  - Agent delegation.
- [ ] Calendar module:
  - Local view.
  - Scheduled agent tasks.
  - Reminders.
- [ ] Goals module:
  - Hierarchy and milestones.
  - Progress tracking.
  - Link to tasks and decisions.
- [ ] Decisions module:
  - Decision log.
  - Reasoning and outcome fields.
- [ ] Knowledge graph:
  - Entity extraction.
  - Graph UI.
- [ ] Business departments as skill/tool presets.

## Out of Scope

- External calendar sync (CalDAV) in MVP.
- Email integration in MVP.

## Definition of Done

A user can take notes, manage tasks and goals, log decisions and explore a knowledge graph. Agents can extract tasks and decisions from conversations.
